# Privacy Policy — Versets & Jardins

**Last updated:** March 2026

## 1. Data Collection
Versets & Jardins does not collect, store, or process any personal data from TikTok or Instagram users.

## 2. TikTok API Usage
This application uses the TikTok Content Posting API solely to publish pre-generated videos. No user data, analytics, or profile information is accessed or stored.

## 3. Local Data
The application stores only video scheduling data locally (SQLite database) on the developer's private machine. This data is not shared with any third party.

## 4. Third-Party Services
- TikTok API — for video publication
- Aladhan.com — for Islamic prayer time data
- Pexels API — for background video footage

## 5. Contact
For any questions: contact via GitHub repository issues.
